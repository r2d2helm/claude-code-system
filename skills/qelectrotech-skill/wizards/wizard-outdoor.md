# Wizard : Installation Exterieure

Assistant pour l'eclairage et les prises exterieures (jardin, terrasse, piscine).

## Etape 1 : Zones exterieures

Demander :
- Quelles zones ? (jardin, terrasse, allee, portail, piscine, abri jardin, garage exterieur)
- Distance depuis le tableau principal

## Etape 2 : Points par zone

Pour chaque zone :
- Points lumineux (nombre, type: applique, borne, projecteur, spot)
- Prises (nombre, usage: tondeuse, outils, recharge VE)
- Automatismes (portail, volets, arrosage)

## Etape 3 : Protection specifique exterieur

Regles strictes :
- IP44 minimum pour tout appareillage exterieur couvert
- IP65 pour appareillage expose aux intemperies
- Differentiel 30mA obligatoire pour tous circuits exterieurs
- Mise a la terre specifique si distance > 30m
- Protection contre la foudre (parafoudre recommande)

### Si piscine :
- Volume 0 : Rien
- Volume 1 (< 2m du bassin) : TBTS 12V uniquement
- Volume 2 (2-3.5m) : Classe II, IP44 minimum
- Liaison equipotentielle du bassin OBLIGATOIRE

## Etape 4 : Dimensionnement

Attention a la chute de tension en exterieur (distances longues) :
- Calculer pour chaque circuit
- Augmenter la section si necessaire
- Envisager un tableau secondaire si jardin eloigne
