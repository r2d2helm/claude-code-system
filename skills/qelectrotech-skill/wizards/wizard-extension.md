# Wizard : Extension de Circuit

Assistant pour ajouter un circuit a une installation existante.

## Etape 1 : Type d'extension

Demander :
- Quel type de circuit ajouter ? (eclairage, prises, dedie, special)
- Quelle piece ?
- Combien de points ?

## Etape 2 : Verification tableau existant

Verifier :
- Place disponible dans le tableau (modules libres)
- Differentiel disponible compatible
- Si pas de place -> proposer extension tableau

## Etape 3 : Dimensionnement

Calculer :
- Section cable necessaire
- Calibre disjoncteur
- Longueur de cable estimee
- Materiel necessaire

## Etape 4 : Mise en oeuvre

Proposer le plan de realisation :
- Elements a acheter
- Schema de cablage
- Points d'attention (chute de tension si longue distance)
- Mise a jour du folio QET correspondant
