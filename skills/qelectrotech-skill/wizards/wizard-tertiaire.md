# Wizard : Installation Tertiaire

Assistant pour creer un projet electrique pour un bureau ou commerce.

## Etape 1 : Informations

Demander :
- Type : Bureau, Commerce, Restaurant, Cabinet medical
- Surface totale
- Nombre de postes de travail (si bureau)
- Puissance souscrite
- Triphase ? (souvent oui en tertiaire)

## Etape 2 : Zones

Definir les zones :
- Bureaux / Open space
- Salle de reunion
- Accueil
- Sanitaires
- Cuisine / Coin cafe
- Local technique
- Reserve / Stock
- Parking / Acces

## Etape 3 : Besoins specifiques tertiaire

- Eclairage : Norme NF EN 12464-1 (300-500 lux selon zone)
- Prises bureaux : 1 bloc 4 prises par poste + 1 RJ45 + 1 VDI
- Baie informatique / local serveur
- Climatisation / CTA
- Enseignes lumineuses (commerce)
- Eclairage de securite (BAES)
- Alarme incendie

## Etape 4 : Dimensionnement

Specifiquement tertiaire :
- Tableau divisionnaire par zone si grande surface
- Compensateur d'energie reactive si > 36 kVA
- UPS/onduleur pour informatique

## Etape 5 : Circuits

Organiser les circuits tertiaires.

## Etape 6 : Tableau

Concevoir le TGBT tertiaire (potentiellement triphase).

## Etape 7 : Generation

Creer le projet QET avec folios par zone.
