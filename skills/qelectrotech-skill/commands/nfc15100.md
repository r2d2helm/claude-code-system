# Verifier la conformite NF C 15-100

Analyse un projet QET et verifie la conformite avec la norme NF C 15-100.

## Actions

1. **Charger le projet** et identifier les pieces (par folio/titre)
2. **Pour chaque piece, verifier** :
   - Nombre minimum de prises respecte
   - Nombre minimum de points lumineux respecte
   - Circuits dedies presents (cuisine, SdB)
   - Volumes SdB respectes (si applicable)
3. **Verifier le tableau electrique** :
   - Presence disjoncteur general
   - Presence differentiel(s) 30mA
   - Au moins 1 differentiel type A
   - Calibres disjoncteurs corrects par type de circuit
   - Sections cables adequates
4. **Verifier les circuits** :
   - Max 8 points par circuit eclairage
   - Max 8 prises par circuit prises
   - Circuits dedies bien separes
5. **Generer le rapport de conformite**

## Verifications par piece

### Sejour (< 28m2)
- [ ] >= 5 prises 16A (1 par tranche de 4m2)
- [ ] >= 1 point lumineux au plafond
- [ ] 1 prise communication (RJ45 ou TV)
- [ ] 1 prise commandee si pas de DCL

### Sejour (> 28m2)
- [ ] >= 7 prises 16A
- [ ] >= 2 points lumineux
- [ ] 2 prises communication

### Chambre
- [ ] >= 3 prises 16A
- [ ] >= 1 point lumineux au plafond
- [ ] 1 prise communication (RJ45)

### Cuisine
- [ ] >= 6 prises (dont 4 au-dessus plan de travail)
- [ ] >= 1 point lumineux
- [ ] Circuit dedie four (20A)
- [ ] Circuit dedie plaques (32A)
- [ ] Circuit dedie lave-vaisselle (20A)
- [ ] Differentiel type A

### SdB
- [ ] Volumes respectes (0, 1, 2, hors volume)
- [ ] Appareillage IP44 en volume 2
- [ ] Circuit dedie lave-linge (20A, type A)
- [ ] Circuit dedie chauffe-eau (20A)
- [ ] Liaison equipotentielle supplementaire

### WC
- [ ] >= 1 prise (si > 4m2)
- [ ] >= 1 point lumineux

### Couloir
- [ ] >= 1 prise 16A
- [ ] >= 1 point lumineux par 4m lineaire

### Exterieur
- [ ] >= 1 point lumineux par acces
- [ ] Appareillage IP44/IP65

## Eclairage tertiaire NF EN 12464-1

Si le projet est de type tertiaire, verifier les niveaux d'eclairement :

| Local | Eclairement min (lux) |
|-------|----------------------|
| Bureau individuel | 500 |
| Bureau open space | 500 |
| Salle de reunion | 500 |
| Accueil / Reception | 300 |
| Couloir / Circulation | 100 |
| Escaliers | 150 |
| Sanitaires | 200 |
| Archives / Stockage | 100 |

## Volumes salle de bain (detail)

```
Volume 0 : Interieur baignoire/receveur douche
           → RIEN sauf TBTS 12V (transfo hors volume)
           → IPX7 obligatoire

Volume 1 : Au-dessus bain/douche jusqu'a 2.25m du sol
           → Eclairage IPX4 (IPX5 si jets) classe II/III
           → Chauffe-eau horizontal instantane admis
           → Interdit : prises, interrupteurs, boites de jonction

Volume 2 : 60 cm autour du volume 1, jusqu'a 2.25m
           → Eclairage classe II, IPX4
           → Appareil de chauffage classe II
           → Prises rasoir 20-50VA via transfo separateur

Hors volume : Au-dela de 60 cm
           → Tout equipement classe I ou II autorise
           → Prise 16A standard autorisee
           → Lave-linge, seche-linge admis (circuit dedie 20A type A)
```

## Format de sortie

```
# Rapport NF C 15-100 - Projet "Nom"

## Score global : XX/100

## Par piece
| Piece | Prises | Eclairage | Dedies | Conformite |
|-------|--------|-----------|--------|------------|
| Salon | 6/5 OK | 2/1 OK | N/A | CONFORME |
| Cuisine | 4/6 NOK | 1/1 OK | 2/3 NOK | NON CONFORME |
...

## Points non conformes
1. Cuisine : 2 prises manquantes au plan de travail
2. Cuisine : Circuit dedie lave-vaisselle absent
3. SdB : Prise en volume 2 sans IP44
...

## Recommandations
...
```

## Exemple

```
/qet-nfc15100 "projet.qet"
/qet-nfc15100 check --rooms "Salon=F1,Cuisine=F2,SdB=F3"
```
