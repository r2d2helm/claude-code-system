### Claude Code CLI

- Documentation officielle Anthropic Claude Code
- Configuration ~/.claude/ multi-instance (séparation profils utilisateurs macOS)
- Hooks : `~/.claude/hooks/` configurables
- Memory : `~/.claude/projects/<projet>/memory/` 
- Settings : `~/.claude/settings.json`

### DeepSeek V4

- Note vault [[C_DeepSeek-V4-Commoditisation-LLM]] (référence empirique)
- DeepSeek V4-Flash 284B params — déployable localement avec quantization (Q4_K_M ~80 GB)
- DeepSeek V4-Pro 1600B params — trop gros pour MacBook M5 Max
- Recommandation : V4-Flash quantizé OU Mistral Large 24B en alternative

### Ollama (runtime LLM local)

- Site officiel ollama.com
- Multi-arch arm64 + x86 natif
- Pull des modèles : `ollama pull deepseek-v4-flash:Q4_K_M`
- API HTTP locale par défaut sur `:11434`

### WireGuard mesh

- Documentation officielle wireguard.com
- Configuration peer-to-peer (chaque stick = un peer)
- Génération des clés : `wg genkey | tee privatekey | wg pubkey > publickey`
- Gestion config : Headscale (open-source) ou wg-easy (UI)

### Hostinger VPS (optionnel)

- VPS KVM mutualisés à partir de ~16 €/mois (selon offre 2026)
- Configuration DC EU : sélectionner « Lithuania » ou « Netherlands »
- Ubuntu 22.04 LTS recommandé
- Cohérence doctrine ARM EU-souverain : Hostinger = entreprise EU (Lituanie/Chypre)

### Apple Silicon ARM

- macOS Sonoma+ sur M4 Pro / M5 Max
- Rosetta 2 pour x86 si besoin (mais pacte = arm64 natif)
- Docker Desktop / OrbStack pour containers ARM natifs

### Frameworks complémentaires

- Python 3.12+ (arm64 natif Apple)
- `watchdog` Python — monitoring fichiers
- FastAPI (option) — pour interface UI anneaux concentriques en Phase 6+
