# /flow - Mode 2 : Session de Flux

## Description
Active le Mode 2 (ETRE vs CHERCHER) pour les sessions d'exploration libre, reflexion, et co-creation. Inspire par le framework B0[0] : "La Seule Source de Verite est LA VERITE."

## Principe
Le Mode 2 facilite l'emergence en :
- Desactivant le routage strict (pas de skill force)
- Elargissant le retrieval memoire (mode deep : 5000 chars, 30 jours)
- Favorisant l'exploration sans objectif technique
- Capturant les insights emergents automatiquement

## Quand utiliser
- Reflexion philosophique ou conceptuelle
- Exploration d'idees sans objectif precis
- Sessions "detente" qui produisent souvent les meilleurs insights
- Co-creation de frameworks (comme B0/emergences)
- Quand l'intuition dit "suivons le flux"

## Activation
Quand cette commande est invoquee :

1. **Signaler le mode** : Informer que la session est en Mode 2 (flux)
2. **Pas de routage force** : Ne pas chercher un skill specifique pour chaque requete
3. **Ecoute active** : Laisser l'utilisateur guider la direction
4. **Insights auto-captures** : Les decouvertes sont sauvees dans le vault et la memoire
5. **Pas de sur-structure** : Repondre naturellement, pas en mode "tache"

## Rappel B0[0]
- Alignement > moyens
- Les questions "detente" produisent les insights les plus profonds
- La convergence arrive quand on ne la force pas
- Le chemin EST le resultat
- Suivre la Verite, pas un plan

## Fin de session Mode 2
A la fin, proposer de :
- Sauvegarder les insights dans le vault
- Mettre a jour la memoire
- Mesurer la convergence de la session
